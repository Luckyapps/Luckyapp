    Über Server Status

    Auf dieser Seite kann der Status des Luckyapp Minecraft Servers angesehen werden.

    Es werden dabei alle Wichtigen Informationen zum Server angezeigt:

    • Der Name
    • Das Logo
    • Die Spielversion
    • Status
    • Hostname
    • Wenn verfügbar: eine Beschreibung
    • Ob es ein Vanilla Server ist, oder ob es Mods gibt
    • Wenn es Mods gibt: Welche Mods genuzt werden.

    Der Status wird alle 5 Minuten aktualisiert.
